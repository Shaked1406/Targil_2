# HW2 - HTML & CSS Project

# Project Description:
This project was created as part of a Web Programming course.
The website was built using several HTML files and several CSS files, where each file represents a different design method.

# Link to the live site:
https://shaked1406.github.io/Targil_2/

# Files Description:

## HTML Files
### index.html - 
Main entry page containing links to all layout versions.
### CSS Plain.html - 
Page layout implemented using plain CSS and floats.
### CSS_Plain_copilot.html -
Same layout implemented with Copilot-generated plain CSS.
### flex_copilot.html-
Layout implementation using CSS Flexbox.
### bootstrap_copilot.html -
Responsive layout built using Bootstrap grid system.

## CSS Files
### CSS Plain.css
Manual CSS styles using floats and percentage-based columns.
### CSS_Plain_copilot.css - 
Copilot-generated CSS using floats for layout.
### flex_copilot.css - 
CSS styles based on Flexbox for responsive layout.
### bootstrap_copilot.css - 
Custom CSS that complements the Bootstrap layout.
