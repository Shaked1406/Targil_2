# Targil_2
# Layouts Exercise

This repository contains four different implementations of the same layout:

1. CSS PLAIN – manual implementation  
2. CSS PLAIN – generated by GitHub Copilot  
3. Flexbox layout – generated by GitHub Copilot  
4. Bootstrap layout – generated by GitHub Copilot  

## How to run

You can open the project both from a desktop computer and from a mobile device.

The live site is available at:  
https://your-username.github.io/targil-layouts/

On that page (`index.html`), you will see four links – one for each implementation.  
Click any link to open the corresponding layout page.
